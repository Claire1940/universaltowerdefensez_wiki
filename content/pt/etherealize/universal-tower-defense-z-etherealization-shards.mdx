<Section icon={Shield} title="Comparação: Eterealização vs. Evolução">

Muitos jogadores confundem Eterealização com Evolução. Embora ambas usem **fragmentos de eterealização de Universal Tower Defense Z** ou materiais semelhantes, elas servem a propósitos diferentes. A Eterealização é uma etapa de aumento de atributos, enquanto a Evolução frequentemente altera completamente o modelo, as habilidades e a raridade da unidade.

Compreender a diferença é fundamental para o planejamento a longo prazo. Geralmente, você precisa Eterealizar uma unidade antes que ela se torne elegível para certas Evoluções de tier alto.

| Recurso | Eterealização | Evolução |
| :--- | :--- | :--- |
| **Objetivo Principal** | Multiplicadores de Atributos | Mudança de Habilidade/Forma |
| **Custo de Recursos** | Fragmentos / Ouro | Fragmentos / Materiais Específicos |
| **Mudança de Raridade** | Não | Sim (Geralmente aumenta) |
| **Pico de Poder** | Linear / Alto | Massivo / Muda o Jogo |
| **Requisito** | Nível de meio de jogo | Drops de Raide específicos |

<Callout type="warning" title="Aviso de Evolução">
Evoluir uma unidade às vezes redefine seu nível atual. Sempre verifique os requisitos específicos de evolução na Wiki para evitar a perda de progresso em seus carregadores principais.
</Callout>

</Section>
