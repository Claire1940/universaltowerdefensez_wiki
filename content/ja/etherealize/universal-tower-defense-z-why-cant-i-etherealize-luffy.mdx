<Section icon={Target} title="ルフィをエーテル化できない一般的な理由">

Universal Tower Defense Zの「ルフィ」ユニットは、公式には**ジョイフル・キャプテン（Joyful Captain）**と呼ばれています。多くのプレイヤーがベースバージョンのキャラクターをエーテル化しようとしますが、それは不可能です。このゲームでは、召喚 -> レベルアップ -> 進化 -> エーテル化という厳格な直線的進行が必要です。エーテル化ボタンがグレーアウトしている、または表示されない場合は、以下の表を参照して不足している条件を特定してください。

| 要件 | ステータス | 説明 |
| :--- | :--- | :--- |
| **進化ステータス** | 必須 | ベースのルフィではなく、**ジョイフル・キャプテン**（進化後）である必要があります。 |
| **ユニットレベル** | レベル100以上 | ほとんどのS+ティアユニットは、エーテル化するために最大または最大に近いレベルが必要です。 |
| **エーテル化の欠片** | 50-100個 | **レイド**または**バーチャルレルム**で専用にドロップします。 |
| **ユニバーサル・ジェム** | 100,000以上 | 変身プロセスの二次通貨として必要です。 |
| **インベントリロック** | 解除済み | ユニットがインベントリで「ロック」されている場合、メニューが無効になることがあります。 |

<Callout type="warning" title="警告">
エーテル化する前に、余剰がない限り**特性リロール（Trait Rerolls）**を使いすぎないようにしてください。エーテル化プロセスによって、特定の特性がユニットの新しいスケーリングとどのように相互作用するかが変わる場合があります。
</Callout>

<Section icon={Steps} title="ジョイフル・キャプテンをエーテル化するためのステップバイステップガイド">

基本的な条件は満たしているのに進めない場合は、以下の手順に従って、ゲームがユニットをアップグレード対象として認識しているか確認してください。これは、「**ルフィをエーテル化できない**」という問い合わせに関連する一般的なエラーを修正する最も確実な方法です。

<Steps>
  <Step num="1" title="進化状態の確認">
    ユニットインベントリに移動し、ルフィを探します。まだベース形態の場合は、**海の断片**と**空の断片**を使用して**ジョイフル・キャプテン**に進化させる必要があります。エーテル化メニューは進化後のユニットにのみ表示されます。
  </Step>
  <Step num="2" title="レベル閾値への到達">
    ジョイフル・キャプテンを装備し、少なくともレベル100に達するまで**ストーリーモード**や**バーチャルレルム**のステージをプレイしてください。一部のプレイヤーからは、特定のレアリティの2026年レベル上限に達しないとエーテル化オプションが解放されないという報告もあります。
  </Step>
  <Step num="3" title="エーテル化の欠片の収集">
    **レイド**ポータルに入るか、**ワールドレイド**に参加してください。特定数のエーテル化の欠片が必要です（現在の必要数は進化NPCで確認してください）。これらはレアなドロップ品であり、複数回の周回が必要になる場合があります。
  </Step>
  <Step num="4" title="エーテル化メニューへのアクセス">
    メインハブに行き、**進化/エーテル化NPC**を見つけます。NPCに話しかけ、リストからジョイフル・キャプテンを選択します。すべての素材が緑色にハイライトされていれば、変身を進めることができます。
  </Step>
</Steps>

<Section icon={Shield} title="トラブルシューティングと一般的なエラー">

すべての素材が揃っていても、テクニカルなバグが発生することがあります。依然として「**ルフィをエーテル化できない**」とお困りの場合は、アップデート4.0でコミュニティが特定した以下のトラブルシューティングを検討してください。

| 問題 | 解決策の候補 |
| :--- | :--- |
| **ボタンが反応しない** | サーバーに再参加するか、ラグを減らすためにプライベートサーバーを試してください。 |
| **素材がカウントされない** | 欠片が**保管庫（Vault）**ではなく**インベントリ**にあることを確認してください。 |
| **ユニットの選択間違い** | 重複しているベースのルフィをクリックしていないか再確認してください。 |
| **通貨のグリッチ** | 残高を「更新」するために、さらに1ジェム獲得する必要がある場合があります。 |

<Callout type="warning" title="重要">
最近ジョイフル・キャプテンに**ステータスリセット**を使用した場合は、サーバーがデータを保存するまで（通常60秒）待ってからエーテル化を試みてください。
</Callout>
