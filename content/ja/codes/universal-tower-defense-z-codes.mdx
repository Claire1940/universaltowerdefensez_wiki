<Section icon={Target} title="環境ユニット向けの特性リロール戦略">

コード特典の中で最も価値があるのは特性リロールです。特性は、射程、ダメージ、または攻撃速度に大幅なバフをもたらします。しかし、リロールは確率（RNG）に基づいているため、手持ちのストックを使い果たさないための戦略が必要です。

コードから50個や100個の特性リロールを受け取ったら、現在の環境でS層（S-Tier）に君臨しているユニットを最優先にするべきです。これらのユニットは、投資に見合うだけの長期的な活躍が期待できます。

| ユニット名 | 役割 | 推奨特性 |
| :--- | :--- | :--- |
| **Joyful Captain** | メインDPS / サポート | Godly / Reaper |
| **Jinoo (Shadow Monarch)** | スケーリングDPS | Sovereign / Ace |
| **The Strongest of Today** | デバッファー / DPS | Ethereal / Divine |
| **King Sailor (Unrivaled)** | 範囲攻撃 / リーダー | Unique / Golden |
| **Bulmo** | 経済 / 資金稼ぎ | Wealthy / Efficient |

<Callout type="info" title="特性のティア">
特性はレアリティによって分類されています。「Godly」が最終目標となることが多いですが、「Ace」や「Speedy」のような一部の「Legendary」特性も、中盤のコンテンツやレイドでは十分に実用的です。
