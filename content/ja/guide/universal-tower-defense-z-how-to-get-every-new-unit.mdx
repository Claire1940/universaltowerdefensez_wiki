<Section icon={Shield} title="進化とエーテル化のルート">

**Universal Tower Defense Z**の「新規」ユニットの多くは、最初から最終形態で召喚することはできません。代わりに、ベースとなるバージョンを召喚し、**進化**および**エーテル化**システムを進める必要があります。これは多くのプレイヤーが詰まるポイントであり、ゲーム終盤の特定のモードでしか手に入らないフラグメントや素材を必要とします。

<Steps>
  <Step num="1" title="ベースユニットの召喚">
    Gemやチケットを使用して、召喚エリアからユニット（例：Jinoo）のベースバージョンを入手します。
  </Step>
  <Step num="2" title="最大レベルまで育成">
    ストーリーモードやインフィニットモードでユニットを使用し、レベル上限まで上げます。ベースのポテンシャルを引き出していないユニットは進化できません。
  </Step>
  <Step num="3" title="進化素材の収集">
    レイドを周回して特定のフラグメント（Ocean、Sky、またはUniversalフラグメント）を集めます。一部のユニットには「Alpha Swords」や「Ruler Souls」が必要です。
  </Step>
  <Step num="4" title="進化とエーテル化">
    進化メニューからユニットを変身させます。エーテル化シャードを使用してステータスをさらに強化し、隠されたパッシブを解放します。
  </Step>
</Steps>

| 進化素材 | 入手場所 | 対象ユニット |
| :--- | :--- | :--- |
| **Universal Fragment** | ワールドレイド | ほとんどのS+ティアユニット |
| **Ocean Fragment** | チャレンジモード | 水属性 / サマーユニット |
| **Sky Fragment** | インフィニットモード (Wave 50+) | ヒルおよびハイブリッドユニット |
| **エーテル化シャード** | ウルトラボス戦 | すべてのエーテル化形態 |

<Callout type="warning" title="リソース管理">
**エーテル化シャード**をBティア以下のユニットに浪費しないでください。進化はコストが高く、素材は貴重です。2026年の終盤コンテンツを攻略するため、ティアリストで**S+**または**S**と評価されているユニットにのみ投資しましょう。
</Callout>

</Section>

<Section icon={Swords} title="特性とリロール：ユニットの最適化">

ユニットを入手するのは半分に過ぎません。もう半分は、正しい**特性**を持っていることを確認することです。「Godly」や「Unique」特性を持つユニットは、ベースバージョンよりも最大50%優れた性能を発揮します。**Universal Tower Defense Z**では、（コードやレイドで入手できる）**特性リロール**を使用して、これらの強力な補正を狙います。

| 特性名 | 効果 | 優先度 |
| :--- | :--- | :--- |
| **Godly** | 大幅なダメージ＆射程ブースト | 最高 |
| **Unique** | ユニット固有の特殊能力 | 高 |
| **Golden** | ゴールド獲得量増加 (ファーム用) | 高 (Bulmoのみ) |
| **Swift** | クールダウン短縮 | 中 |

<Callout type="info" title="ステータスロック">
リロールを行う際は、**ステータスロック**を使用して高ティアのロールを保護しながら、より良いステータスを追い求めましょう。次のアップデートで入れ替える予定のユニットには、決してステータスロックを使用しないでください。
</Callout>

</Section>
