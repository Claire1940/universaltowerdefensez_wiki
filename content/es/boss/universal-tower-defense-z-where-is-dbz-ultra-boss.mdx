<Section icon={Target} title="Mecánicas y Recompensas del Jefe">

El Ultra Boss de DBZ presenta múltiples fases, mecánicas de escudo y ataques de área de efecto de alto daño que pueden desactivar temporalmente tus torres. Comprender estas mecánicas es la diferencia entre una victoria exitosa y una derrota repentina.

- **Fase de Escudo**: Al 50% de salud, el jefe obtiene un escudo temporal. Concéntrate en ataques de alta frecuencia para romper el escudo rápidamente.
- **Modo Furia**: Por debajo del 30% de salud, el jefe se mueve un 25% más rápido. Guarda tus habilidades de aturdimiento específicamente para esta fase para evitar fugas.
- **Ataques de Aturdimiento**: El jefe desactivará periódicamente las unidades cercanas. Distribuye tus atacantes para evitar que toda tu defensa sea desactivada a la vez.

| Objeto de Recompensa | Tasa de Obtención | Uso Principal |
| :--- | :--- | :--- |
| **Fragmentos de Eterealización** | Alta (Garantizada) | Se usa para eterealizar unidades y aumentar las estadísticas base |
| **Giros de Atributo (Trait Rerolls)** | Media | Cambia los atributos de las unidades para obtener modificadores de combate óptimos |
| **Materiales de Evolución Exclusivos de DBZ** | Baja | Requeridos para evolucionar unidades de anime específicas de alto nivel |
| **Gemas Universales** | Media | Se gastan en la tienda del evento para obtener recompensas por tiempo limitado |

<Callout type="tip" title="Consejo de Farmear">
Equipa a tus unidades con Reliquias de soporte que reduzcan la duración del aturdimiento. Esto contrarresta directamente los ataques de desactivación del jefe y mantiene constante tu producción de DPS.
</Callout>
